## 1. Information Architecture School (01-04)
> Philosophy: "Data is not decoration, it is building material"

### 01. Pentagram - Michael Bierut style
**Philosophy**: Font is language, grid is thought
**Core Features**:
- Extremely restrained colors (black and white + 1 brand color)
- A modern interpretation of the Swiss grid system
- Typography as the primary visual language
- Strategic use of negative space (60% + white space)

**Prompt word DNA**:
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Masterpiece**: Hillary Clinton 2016 campaign identity
**Search keywords**: pentagram hillary logo system

## Instructions for using prompt words

**Composition formula**: `[style prompt word DNA] + [scene template (see scene-templates.md)] + [specific content]`

### Core Principle: Describe mood, not layout (Mood, Not Layout)

The key to AI image generation: short prompt words > long prompt words. Three sentences describing the mood and content are more effective than 30 lines of layout details.

| Writing that kills diversity | Writing that inspires creativity |
|----------------|----------------|
| Specify the color ratio (60%/25%/15%) | Describe the mood ("warm like Sunday morning") |
| Specify layout position ("title in center, right of image") | Reference specific aesthetics ("Pentagram editorial feel") |
| Limit character poses and expressions | Let AI interpret style naturally |
| List all visual elements to be included | Describe what the audience should feel |

### Good / Bad Example

**Bad — Over-constrained (AI generated empty and flat):**
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good — Emotionally driven (generation is diverse and textured):**
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Execution path selection

Select from the "Best Path" column of the cheat sheet:
- **AI generated**: style with clear visual elements (06/07/12/13/14/15/16/20), straight out of Gemini/Midjourney
- **HTML rendering**: style that relies on precise typography (01/03/04/10/11/17/18), code controls data and layout
- **Hybrid**: HTML for skeleton layout + AI to generate images/background (02/05/08/09/19)

### Quality Control

1. ❌ Don’t write “in the style of Pentagram” directly → ✅ Use specific design features to describe
2. Text often goes wrong during AI generation → Replace text after generation
3. The ratio is prone to distortion → clearly specify the aspect ratio
4. First generate 3-5 variants, select the best and then refine
