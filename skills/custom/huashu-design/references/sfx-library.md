## Usage specifications

### Volume recommendations (from apple-gallery-showcase.md audio dual-track system)
- **SFX Master Track**: `1.0` (no attenuation)
- **BGM background track**: `0.4 ~ 0.5` (SFX obviously penetrates)
- **Multiple SFX overlays**: Use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg splicing template
```bash
# Single SFX alignment time points:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection decision tree
1. **With tactile action** (typing/clicking/sliding) → `keyboard/` or `ui/`
2. **Element entry/exit** → `transition/`
3. **Container layer operations** (card/modal) → `container/`
4. **Status feedback** (success/failure/notification) → `feedback/`
5. **Progress/time lapse** → `progress/`
6. **Brand placement/important moment** → `impact/`
7. **AI magic/transformation** → `magic/`
8. **Command line/code demonstration** → `terminal/`

### Avoid overdubbing accumulation
- `max 2 SFX` concurrently at the same point in time
- 3 can be placed when the BGM drops below 0.3
- Clear other SFX during brand impact (leave blank for 0.2s before placing the point)

## Prompt writing principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three elements of a good prompt**:
1. **Physical description of sound**: What object, what action ("mechanical keyboard single key press")
2. **Texture/Style Limited**: apple-style / ios-style / cinematic / retro
3. **Counterexample exclusion**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
